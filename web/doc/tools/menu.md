# 设计模板 #

大型项目功能复杂，设计页面动辄上百，耗费人力资源多，关键很多页面重复度较高，但开发周期密集。设计模板旨从使用场景角度出发，将历史项目中的通用架构（基本布局、视觉流、交互流程、设计经验等）提炼成一套套设计模板。当设计师再次设计类似功能（如登录功能），无需事无巨细的重新考虑，直接使用（登录）模板。不但能大量缩短设计时间，也能在较短的设计周期保证设计质量，亦能避他人设计之缺陷，扬自己设计所长。
